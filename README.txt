Getting Started
To work with this repository locally:

Clone the repository:

bash
git clone https://github.com/deckerd451/charlestonhacks.github.io.git
Open the project folder and serve it locally. You can use an extension like Live Server in VSCode, or run:

bash
python -m http.server
Then visit http://localhost:8000 in your browser.

Make your changes and push updates to your fork or branch.
